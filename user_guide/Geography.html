Geography#

Datashader is a general-purpose tool for rasterizing (and re-rasterizing) data of many different types. To make it easier to apply this general functionality to the particular domain of geoscience, there are some geospatial-specific utilities that can be used:

  • Surface toolset

  • Proximity (Distance) toolset

  • Zonal Statistics

  • Classification toolset

  • Remote Sensing

  • Pathfinding

These functionality is provided in the xarray-spatial library. You can check out its example notebooks to see how to use the functions.

See also GeoViews, which is designed to work with Datashader to provide a large range of additional geospatial functionality.
